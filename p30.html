เวลาชีวิตของตัวแปรในฟังก์ชัน

ของตัวแปรใน ที่เป็น ในฟังก์ชันจะถูกจัดสรรพื้นที่เมื่อฟังก์ชันถูกเรียก และพื้นที่นั้นจะถูกคืนให้กับหน่วยความจำเมื่อการรันโปรแกรมออกจากฟังก์ชัน โค้ดฟังก์ชันที่เขียนด้วยภาษา C ที่มี x, y และ result ทั้งสามตัวแปรจะถูกจัดสรรเมื่อฟังก์ชันถูกเรียกใช้งาน และเมื่อมีการรีเทิร์นออกจากฟังก์ชัน พื้นที่ที่ถูกจัดสรรให้กับตัวแปรเหล่านั้นก็จะถูกคืน (deallocated) ให้กับหน่วยความจำ ดังนั้นของตัวแปรในฟังก์ชันจึงเริ่มที่บรรทัดที่มีการประกาศตัวแปร และสิ้นสุดเมื่อรีเทิร์นออกจากฟังก์ชัน
ตัวอย่างที่ 1 
/* main unit */
        ...
        computesum(a, b);
        computesum(c, d);
        /* function unit */
        int   computesum(int x, int y)
        {
               int result;
               result  = x +y;
               return result;
         }

อธิบายโค้ด


หมายเหตุ
ให้กับตัวแปร x และ y เกิดขึ้นเมื่อมีการสำเนาค่าจากตัวแปรที่เป็น Argument (a,b,c,d) การจัดสรรพื้นที่ให้กับ result เกิดขึ้นเมื่อมีการประเมินผลนิพจน์ x+y แล้วกำหนดค่า ตัวแปร x,y, result เป็น ซึ่งจะถูกจัดสรรพื้นที่ใหม่ถูกครั้งที่ฟังก์ชันถูกเรียกและมีการคืนพื้นที่ทุกครั้งก่อนรีเทิร์นออกจากฟังก์ชัน

เวลาชีวิตของตัวแปรสตาติกในฟังก์ชัน
เวลาชีวิตของตัวแปรสตาติกในฟังก์ชัน จะเริ่มขึ้นเมื่อตัวแปรสตาติกถูกจัดสรรพื้นที่ในหน่วยความจำจากการเรียกฟังก์ชันครั้งแรก (ตำแหน่ง/addressได้ตั้งแต่เวลาคอมไพล์) และจะสิ้นสุดลงเมื่อโปรแกรมรันเสร็จ
ตัวอย่างที่ 2

/* main unit */
        ...
        computesum(a, b);
        computesum(c, d);
        
        /* function unit */
        int   computesum(int x, int y)
        {
               static int result  = 0;
               result  = result + x + y;
               return result;
         }

อธิบายโค้ด


หมายเหตุ
การจัดสรรพื้นที่ให้กับตัวแปร x และ y เกิดขึ้นเมื่อมีการสำเนาค่าจากตัวแปรที่เป็น Argument (a,b,c,d) เพราะ result เป็น การจัดสรรพื้นที่ให้กับ result จะเกิดขึ้นครั้งแรกเมื่อฟังก์ชัน computesum ถูกเรียกครั้งแรกและจะยังไม่ถูกคืนพื้นที่ โดยพื้นที่นั้นจะถูกคืนเมื่อรันโปรแกรมจบ ซึ่งต่างจากตัวแปร x และ y ซึ่งเป็น ที่มีการจัดสรรใหม่ทุกครั้งที่ฟังก์ชันถูกเรียกและมีการคืนพื้นที่เมื่อริเทิร์นออกจากฟังก์ชัน

ตอบคำถามต่อไปนี้

ตัวแปรประเภทใดในฟังก์ชันจะถูกจัดสรรพื้นที่ใหม่ทุกๆครั้งที่ฟังก์ชันถูกเรียกและพื้นที่นั้นถูกคืนให้กับหน่วยความจำทุกๆครั้งที่รีเทิร์นออกจากฟังก์ชัน

พื้นที่ๆถูกใช้งานในขณะเรียกฟังก์ชันของตัวแปรประเภทใดไม่มีการคืนพื้นที่เมื่อรีเทิร์นออกจากฟังก์ชัน
