คือ เวลาที่สามารถอ้างอิงตัวแปรไปยังตำแหน่งหนึ่ง ๆ ในหน่วยความจำที่ตัวแปรผูกมัดอยู่ เวลาชีวิตของตัวแปรมีความสัมพันธ์กับช่วงเวลาตั้งแต่ สำหรับตัวแปรไปจนถึงเวลาที่เกิด ดังแสดงในภาพ เมื่อ X ถูกประกาศชนิดข้อมูลดังนั้นพื้นที่ในหน่วยความจำจะถูกจัดสรรให้กับตัวแปร X ซึ่งมีขนาดเท่ากับจำนวนไบต์ที่จัดเก็บเลขจำนวนเต็ม เมื่อโปรแกรมรันจบพื้นที่ที่ถูกจัดสรรนั้นจะถูกคืนให้กับหน่วยความจำ

การจัดสรรพื้นที่ให้กับตัวแปร อาจเกิดขึ้นในเวลาคอมไพล์ สำหรับการประกาศตัวแปรบางประเภท ส่วนใหญ่เกิดขึ้นในเวลารันเมื่อมีการใช้งาน เช่น จากการกำหนดค่า (assignment) และการเรียกฟังก์ชันจัดสรรพื้นที่ (เช่นในภาษา C เรียกฟังก์ชัน ) เกิดเมื่อ

  • (พื้นที่ที่ถูกจัดสรรให้กับตัวแปรโลคอลในฟังก์ชันทั้งหมดจะถูกคืน)
  • เช่น ในภาษา C การเรียกฟังก์ชัน free( ) เป็นการยกเลิกการใช้งานพื้นที่ในหน่วยความจำ
