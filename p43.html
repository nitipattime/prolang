Ordinal Type

คือ ชนิดข้อมูลที่มีการกำหนดช่วงค่าข้อมูล(จำนวนเต็มบวก)ที่เป็นไปได้ ค่าที่เป็นไปได้นั้นถูกกำหนดด้วย Named constant (ค่าคงที่ที่ถูกกำหนดด้วยชื่อ) ตัวอย่างชนิดข้อมูลแบบ Ordinal type ได้แก่ enumeration และ subrange

ตัวอย่างใน C#

enum days {mon, tue, wed, thu, fri, sat, sun};

Named constant อาทิ mon, tue, wed, thu, fri, sat, sun ซึ่งสัมพันธ์กับค่าคงที่คือ 1, 2, 3, 4, 5, 6,7 หรือหากภาษา C ไม่มีชนิดข้อมูลแบบ Enumeration แต่ต้องการกำหนดข้อมูลข้างต้น อาจจะต้องเขียนคำสั่งดังนี้ 

int days;
        
        int mon  = 1, tue = 2, wed = 3, thu = 4, fri = 5, sat = 6, sun = 7;

และเมื่อต้องการนำไปใช้ อาจเขียนคำสั่งดังนี้

if (days == mon)
        
           ...
        
        else
        
        .....

การออกแบบควรคำนึงถึง เรื่องดังต่อไปนี้ จะอนุญาตให้กำหนด named constant ที่ซ้ำกันได้หรือไม่ เช่น 

enum days {mon, tue, wed, thu, fri, sat, sun};
        
        enum businessdays {mon, tue, wed, thu, fri};

ค่าของ Enumeration จะต้องแปลงเป็นชนิดข้อมูลใด เช่น เป็นเลขจำนวนเต็มบวก ค่าของ Enumeration เป็นอย่างไร เช่น เริ่มที่ 0 หรือ 1

ข้อดีของ Enumerated Type

ช่วยให้การอ่านโค้ดเข้าใจได้ง่าย ไม่จำเป็นต้องเขียนโปรแกรมด้วยการแทนหมายเลข เช่น 

  enum colors {red, blue, green, yellow, black};
        
          colors myColor = blue, yourColor = red;

colors คือตัวแปรที่เป็น Enumeration type Compiler สามารถตรวจสอบความถูกต้องของตัวแปรที่เกี่ยวข้องกับ Enumeration type ได้ เช่น จะเกิด error เมื่อเขียนคำสั่ง 

colors myColor = silver;    //silver ไม่ใช่ named constant ของ colors

ชนิดข้อมูลแบบ Subrange (ซับเรนจ์) เช่น 12..18 เป็นซับเรนจ์ของชนิดข้อมูลแบบอินเทเจอร์ ตัวอย่างในภาษา Ada 

type Days is (mon, tue, wed, thu, fri, sat, sun);
        
        subtype Weekdays is Days range mon..fri;
        
        subtype Index is Integer range 1..100;
        
        Day1 : Days;       //ตัวแปรชื่อ Day1
        
        Day2 : Weekday;    //ตัวแปรชื่อ Day2

ข้อดีคือ ช่วยให้การอ่านโค้ดง่าย มีความเชื่อถือได้ เช่น คอมไพเลอร์สามารถตรวจสอบคำสั่งที่ไม่ถูกต้องได้ เช่น 

Day2 := Day1;      //Day2 เป็น Weekday ซึ่งเป็น subtype ของ Day1 เป็น Days ดังนั้นการกำหนดค่า Days ซึ่งเป็นไทป์ที่ใหญ่กว่าให้กับ subtype จะไม่
        
                           //สามารถทำงานได้

ถือว่าเป็นการกำหนดค่าที่ไม่ถูกต้อง

หรือ

Ragged (Jagged) array คือ อาเรย์ที่มีได้มากกว่า 1 มิติ โดยแต่ละมิติอาจมีขนาดแตกต่างกัน

ตัวอย่างใน C#:

int[][] jaggedArray = new int[3][];
        
        jaggedArray[0] = new int[1];
        
        jaggedArray[1] = new int[2];
        
        jaggedArray[2] = new int[3];

ตัวอย่างใน Java:

 int[][] jaggedArray = new int[3][];             //create three null int[] elements
        
          for (int i = 0; i < jaggedArray.Length; i++) { // Length 3 (rows)
        
                       jaggedArray[i] = new int[i+1];    // each row is a different length
        
             }

ดัชนีอ้างอิงอาเรย์ (Array Indexing)

ดัชนีอ้างอิง (Indexing or subscripting) คือการจับคู่ระหว่างตัวชี้กับสมาชิกในอาเรย์

ข้อควรพิจารณาในการออกแบบดัชนีอ้างอิง

  • จะใช้สัญลักษณ์ใดในการกำหนดดัชนีอ้างอิง ภาษาการโปรแกรมอาจใช้สัญลักษณ์ที่แตกต่างกันไปในการกำหนดดัชนีอ้างอิง เช่น อาจใช้ [], {}, หรือ ()
    ตัวอย่าง Ada 
     Sum := Sum + B(I)

    ตัวอย่าง PHP 
     $p = array(“Hello”, “world”);
        
                     echo $p[0]. “ ”. $p[1];
  • ชนิดข้อมูลของดัชนีอ้างอิง ควรใช้ประเภทใด เช่น ในภาษา Fortran และ C ใช้อินเทเจอร์เท่านั้น ส่วนภาษา Ada ใช้อินเทเจอร์และ enumeration ส่วน Java ใช้อินเทเจอร์
  • ในการคอมไพล์จะต้องทำการตรวจสอบช่วงค่าของดัชนีอ้างอิงหรือไม่
  • การกำหนดค่าเริ่มต้นให้กับอาเรย์ จะดำเนินการอย่างไร บางภาษากำหนดในช่วงของการจัดสรรพื้นที่ (คือมีการกำหนดค่า) เช่น C, C++, Java, C#
    int list [] = {4, 5, 7, 83} ;
        
         C กับ C++
        
        char name [] = ″freddie″;
        
        char *names [] = {″Bob″, ″Jake″, ″Joe″];

    Java สร้างอาเรย์ซึ่งเป็น String objects 
    String[] names = {″Bob″, ″Jake″, ″Joe″};

Heterogeneous Arrays

คืออาเรย์ที่สมาชิกไม่จำเป็นต้องเป็นชนิดข้อมูลแบบเดียวกัน เช่น ในภาษา PHP, Perl, Python, JavaScript, และ Ruby PHP 

$myarray = {10, “Hello”, true,  10.56} ;

การดำเนินการกับอาเรย์

  • ภาษา APL มีตัวดำเนินการกับอาเรย์ของ Vector และเมทริกซ์ รวมทั้งมี Unary operator
  • ภาษา Ada มี array assignment และ catenation
  • ภาษา Python มี array assignments และ catenation
  • ภาษา Ruby มี array catenation

Rectangular และ Jagged Arrays

Rectangular array คือ อาเรย์หลายมิติ ที่แถวแต่ละแถวจะต้องมีจำนวนสมาชิกเท่ากัน และคอลัมภ์แต่ละคอลัมภ์จะต้องมีจำนวนสมาชิกเท่ากัน

Jagged array คือ อาเรย์ที่แถวแต่ละแถวอาจมีจำนวนสมาชิกไม่เท่ากัน

ภาษา C, C++ และ Java, C# ต่างสนับสนุนการสร้าง jagged arrays Fortran, Ada และ C# สนับสนุน Rectangular arrays

Slices

Slice คือ โครงสร้างย่อย (substructure) ของอาเรย์

Slice เป็นประโยชน์ในการดำเนินการ (operation) กับอาเรย์

ตัวอย่างในภาษา JavaScript

var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
        
        var citrus = fruits.slice(1,3);

ผลลัพธ์ที่ได้คือ Orange,Lemon ตัวอย่างในภาษา Python 

vector = [2, 4, 6, 8, 10, 12, 14, 16]
        
        mat = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

ตัวอย่างการอ้างอิง Slice

vector (3:6) คือ vector[3] ถึง vector[5]

mat[0][0:2] คือ สมาชิกตัวแรก และตัวที่สองของแถวแรกของอาเรย์ mat

การอ้างอิงตำแหน่งในหน่วยความจำของอาเรย์

Single-dimensioned arrays:
           address(list[k]) = address (list[lower_bound])  + ((k-lower_bound) * element_size)
        
Multi-dimensioned Arrays:
        Location (a[i,j]) = address of a [row_lb,col_lb] + (((i - row_lb) * n) + (j - col_lb)) * element_size

ตัวอย่างเมื่อ Array มี n = 10 และเก็บ Integer ขนาด 4 bytes เช่น Location (a[2,3]) = address of a[0,0] + (((2 - 0) * 10) + (3 - 0)) * 4 ตัวอย่างโคดในการหาตำแหน่งของอาเรย์ [i,j]
