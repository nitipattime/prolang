ประเมินด้านการเขียน

ความสามารถในการเขียนของภาษาการโปรแกรม

จะพิจารณาการใช้คำสั่งในการแก้ปัญหา ภาษาการโปรแกรมที่มีความสามารถในการเขียนที่ดีนั้นต้อง ทำให้เขียนโปรแกรมได้ง่าย เช่นมีปัญหาแล้วสามารถเขียนคำสั่งแก้ปัญหาได้ง่ายกว่า ทำให้การเขียนโค้ดไม่ยาวเกินไป การที่โค้ดสั้นส่งผลดีในการเกิดข้อผิดพลาดได้น้อย มีคำสั่งที่นักเขียนโปรแกรมนำไปใช้ในการเขียนโปรแกรมเพื่อแก้ปัญหาที่ซับซ้อนได้ง่าย เขียนโปรแกรมได้ง่ายยกตัวอย่างเช่น ในภาษา C นั้น 

count++;
เขียนได้ง่ายกว่า 
count = count + 1

อาจกล่าวได้ว่าภาษาที่่มีคุณลักษณะ (การเขียนคำสั่งทำได้หลากหลายวิธี เช่น การใช้ Increment/Decrrement operator) นั้นทำให้เขียนง่าย แต่ในทางกลับกันเกิดข้อเสียที่อาจจะทำให้อ่านโค้ดได้ยาก หรือการใช้ 

x = (y > 10) ? 20: 50;

เขียนได้ง่ายกว่า

if (y > 10)
             x = 20;
        else
             x = 50;

อธิบายโค้ด

ตอบคำถามต่อไปนี้

ภาษาโปรแกรมมีคุณลักษณะใดที่กำหนดให้ตัวดำเนินการซึ่งทำให้เขียนโปรแกรมได้ง่าย

ภาษาโปรแกรมที่มีคำสั่งเงื่อนไขแบบใดที่สนับสนุนความสามารถในการเขียน
