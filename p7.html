ภาษาการโปรแกรมในกระบวนทัศน์เชิงบังคับ

เป็นวิธีเขียนโปรแกรมที่ประกอบด้วยกลุ่มคำสั่งที่เกี่ยวข้องกับการเปลี่ยนแปลงสถานะ (state) และสถาปัตยกรรมของคอมพิวเตอร์ Von Neumann ซึ่งมีองค์ประกอบคือ และ ดามภาพ



วิธีการเขียนโปรแกรมในลักษณะนี้มีขั้นตอนการทำงานที่สอดคล้องกับการทำงานในสถาปัตยกรรมแบบ Von Neumann กล่าวคือ

  1. จะเกี่ยวข้องกับหน่วยความจำ ซึ่งตัวแปรนั้นจะอ้างอิงค่าข้อมูลที่เก็บในหน่วยความจำ
  2. เช่น คำสั่งให้ทำเงื่อนไขทดสอบ หรือการกระโดด เกี่ยวข้องกับ ในหน่วยประมวลผลซึ่งทำหน้าที่ตีความคำสั่งและควบคุมลำดับการดำเนินการคำสั่ง
  3. เกี่ยวข้องกับ และการจัดเก็บค่าข้อมูลในหน่วยความจำ
  4. เกี่ยวข้องกับการประมวลผลในหน่วยประมวลผลทางคณิตศาสตร์และตรรกะ (ALU) ซึ่งเป็นส่วนประกอบในหน่วยประมวลผล

การเปลี่ยนแปลงสถานะ จึงอาจเทียบได้กับการเปลี่ยนแปลงค่าข้อมูลในหน่วยความจำ หรือค่าของตัวแปรนั่นเอง ตัวอย่างโปรแกรมที่เขียนด้วยภาษา 

        program gcd( input, output );
        
        var i, j : integer;
        
        begin
        
            read( i, j );
        
            while i <> j do
        
               if i > j then i := i - j
        
               else j := j - i;
        
            writeln( i );
        
        end.
