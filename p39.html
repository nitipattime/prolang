  2. รูปแบบการเขียนประโยคคำสั่งที่เกี่ยวข้องกับเงื่อนไขเป็นแบบ Single statement หรือ Multiple statements
  3. สามารถเขียนในลักษณะที่ซับซ้อนมากขึ้นได้หรือไม่ เช่น Nested if

ตัวอย่างการเขียนคำสั่ง if-else ในภาษา C

ปกติในภาษา C นั้นคำสั่ง if-else จะมี Single statement ที่สอดคล้องกับเงื่อนไข ดังนั้นจึงไม่ต้องกำกับด้วยเครื่องหมายปีก็มีการผูกมัดกับค่าข้อมูลด้วยเครื่องหมายปีกกา 
 if(a > 0)
             a++;
         else
             a--;

การเขียนคำสั่ง if-else ซึ่งสามารถใช้แทนคำสั่งการเลือกแบบหลายทางเลือกหรือ switch...case 

if(a <0 ) {
             x = 0;
         } else if(a==0) {
             x = 1;
         } else if(a>0 && a <10) {
             x = 2;
         } else  {
            x = 9;
         }

ในกรณีที่มีคำสั่งหลายคำสั่งที่สอดคล้องกับเงื่อนไข จะต้องกำกับด้วยเครื่องหมายปีกกา 

if(a>0) {
             a++;
             b = a;
         } else {
             a--;
             b = a;
         }

ตัวอย่างการเขียนคำสั่ง if-else ในภาษา Python ซึ่งโดยปกติแล้วใช้การย่อหน้ากำหนดขอบเขต ดังนั้น if-else ในภาษา Python จึงเป็น Multiple Statements

if balance > 0:
            balance = -transfer
            tranferStatus = true
        if temperature > 70:
            print('Wear shorts.')
        else:
            print('Wear long pants.')

คำสั่งข้างต้นมีคำสั่ง if-else จำนวน 2 คำสั่งเป็นอิสระจากกัน โดยคำสั่งแรก if balance > 0: นั้นไม่มีส่วนของ else

หากกำหนดเยื้องย่อหน้าของคำสั่งใหม่

if balance > 0:
            balance = -transfer
            tranferStatus = true
            if temperature > 70:
                 print('Wear shorts.')
            else:
                 print('Wear long pants.')

คำสั่ง if temperature > 70: จะเป็น Nested if ของ if balance > 0:

ในภาษา Python สามารถใช้คำสั่ง if-elif ซึ่งเทียบเท่าคำสั่ง if...else if... 

if b > a:
          print("b is greater than a")
        elif a == b:
          print("a and b are equal")
        else:
          print("a is greater than b")

หรือสามารถเขียนในรูปแบบคำสั่งย่อ (Shorthand if-else) เช่น 

print("A") if a > b else print("B")

ซึ่งมีความหมายเท่ากับการเขียนคำสั่งในรูปแบบเต็มคือ

 if a > b:
            print("A")
         else
            print("B")

ภาษา Python สามารถเขียนคำสั่งได้ง่าย (มีการย่อคำสั่ง) แต่ก็ทำให้อ่านได้ยากขึ้น เช่น 

print("A") if a > b else print("=") if a == b else print("B")

เทียบเท่ากับการเขียนโค้ดต่อไปนี้ 

if a > b:
          print("A")
        elif  a==b :
          print("=")
        else
          print("B")

Python มีคำสั่งแบบมีทางเลือกแบบซ้อน (Nesting selector) ตัวอย่าง

if control_expression
          if control_expression
          then clause  else clause
          then clause  else clause

สิ่งที่ควรพิจารณาคือการตีความหมายของประโยคคำสั่ง จะต้องยึดถือย่อหน้าเป็นหลัก เช่น คำสั่งด้านล่างนี้จะมีการตีความแตกต่างกัน 

if (sum == 0):
               if (count == 0):
                   result = 0
        else:
            result = 1
        
    if (sum == 0):
                   if (count == 0):
                       result = 0
                   else:
                       result = 1

การกำหนดคำสั่คำสั่งแบบมีทางเลือกสองทางในภาษา Java มีลักษณะเดียวกันกับในภาษา C ในการตีความพิจารณาคำสั่งที่สอดคล้องกับเงื่อนไขและการกำหนดบล็อก แต่ไม่สนใจการเยื้องย่อหน้า เช่นตัวอย่างต่อไปนี้จะมีการตีความแบบเดียวกัน

if (sum == 0)
               if (count == 0)
                   result = 0;
               else result = 1;
        
    if (sum == 0)
                   if (count == 0)
                       result = 0;
            else result = 1;
ซึ่งมีความหมายเช่นเดียวกับ 
    if (sum == 0){

if (sum == 0) {

คำสั่งแบบมีทางเลือกหลายทาง

ภาษาการโปรแกรมส่วนใหญ่มีคำสั่ง switch...case ซึ่งกำหนดรูปแบบคำสั่งได้ดังต่อไปนี้

switch  (X1 ){
          case ( X2 ):  .....statement X3...
         case ( X2):.....statement X3...
        default  X4: .....statement X3....
        };

สิ่งที่ต้องพิจารณาในการออกแบบคำสั่งประเภทนี้ คือ

  1. นิพจน์ที่เกี่ยวข้องกับตัำแหน่ง X1 และ X2 จะสามารถกำหนดได้อย่างไร
  2. ประโยคคำสั่ง ในตำแหน่ง X3 นั้น เป็น รูปแบบใดได้บ้าง เช่น Single statement หรือ Multiple statement และสามารถใช้คำสั่งที่เกี่ยวข้องกับการควบคุมการกระโดด (เช่น break, continue) ร่วมด้วยได้หรือไม่
  3. จะสามารถกำหนดดีฟอลต์ (default) ได้หรือไม่ ดีฟอลต์ หมายความว่าหากไม่สอดคล้องกับเงื่อนไขใดๆในแต่ละเคสข้างต้น

อธิบายโค้ด


ตัวอย่างรูปแบบคำสั่ง switch...case ในภาษา C 

switch( expression )
        {
          case value-1:
              Block-1;
              Break;
          case value-2:
              Block-2;
              Break;
          case value-n:
              Block-n;
              Break;
          default:
              Block-1;
              Break;
        }

ข้อสังเกตในภาษา C กำหนดให้มีการใส่คำสั่ง Break ในส่วนท้ายของแต่ละ case เพื่อป้องกันการพิจารณาไปยังเคสถัดไป คำสั่งในแต่ละกรณีมีได้หลายคำสั่ง และนิพจน์จะเป็นตัวแปรที่มีค่า (value) เป็นเลขจำนวนเต็มบวกหรือลบ และอาจเป็นอักขระ หรือสตริงได้เท่านั้น

ตัวอย่างคำสั่งแบบมีทางเลือกหลายทางในภาษา Ruby
ซึ่งใช้นิพจน์กำหนดเงื่อนไขในการเลือก

           when count < 10 then bag1 = true
           when count < 100 then bag2 = true
           when count < 1000 then bag3 = true
         end
