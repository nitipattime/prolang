ความเท่าเทียมของชนิดข้อมูล

มี 2 ประเภท

  1. คือตัวแปรที่ประกาศในคำสั่งประกาศชนิดข้อมูลเดียวกัน เช่น int a, b, c; // a, b, c จะมีชนิดข้อมูลแบบเดียวกันนั่นคือ Integer เพราะถูกประกาศในคำสั่งเดียวกัน

    อธิบายโค้ด


  2. คือ ตัวแปรที่มีโครงสร้างแบบเดียวกัน เช่น 
    TYPE T1 = RECORD
                     a:INTEGER;
                     b:ARRAY [0..10] OF CHAR;
                  END;
        TYPE T2 = RECORD
                     c:INTEGER;
                     d:ARRAY [0..10] OF CHAR;
                  END;
    โครงสร้างแบบเดียวกันนั้น ให้พิจารณาที่ชนิดข้อมูลของสมาชิก ชื่อตัวแปรต่างกันไม่เป็นไร

    อธิบายโค้ด



การทำ operation บางอย่างของตัวดำเนินการ นั้นตัวแปรที่เกี่ยวข้องอาจจะไม่จำเป็นที่จะต้องมีชนิดข้อมูลแบบเดียวกันเสมอไป ดังนั้นในการคอมไพล์ภาษาการโปรแกรมอาจมีการตรวจหาชนิดข้อมูลที่เข้ากันได้ ซึ่งในภาษาการโปรแกรมหนึ่งๆจะต้องทำการกำหนด compatibility rules ซึ่งระบุชนิดข้อมูลของตัวถูกดำเนินการที่ยอมรับได้ในการทำการดำเนินการหนึ่งๆ 

  int a =10 ; float b = 4553.33, c;
          c  = b + a;

การดำเนินการบวกสามารถกระทำได้กับตัวแปรที่มีชนิดข้อมูลเป็น Integer และ Float ดังนั้นการบวกInteger และ Float สามารถดำเนินการได้เพราะมีความเข้ากันได้ของชนิดข้อมูล อย่างไรก็ตามเรื่องที่จะต้องพิจารณาคือ การเก็บค่าผลลัพธ์นั้น จะมีชนิดข้อมูลเป็นอย่างไร ซึ่งขึ้นกับการออกแบบของภาษาการโปรแกรมว่ามีการจัดการ Implicit conversion อย่างไร เช่น เป็น Narrowing conversion หรือ Widening conversion

ข้อควรคำนึงถึง ในภาษาที่มีการออกแบบ
Subtype จะไม่เท่ากับ Type เช่นในตัวอย่างภาษา Ada

type Color  is (White, Red, Yellow, Green, Blue, Brown, Black);
        subtype Rainbow   is Color range Red .. Blue;

Type checking คือ กระบวนการเพื่อตรวจสอบว่าโอเปอแรนด์เป็นชนิดข้อมูลที่เข้ากันได้ (Compatible type) กับโอเปอเรเตอร์ เช่น หากโอเปอเรเตอร์ + สามารถใช้ในการดำเนินการกับ Integer และ Float ดังนั้นโอเปอแรนด์ทั้งหมดที่เกี่ยวข้องกับโอเปอเรเตอร์ + จะต้องมีชนิดข้อมูลแบบ Integer และ Float เท่านั้น หากโอเปอแรนด์ไม่ใช่ชนิดข้อมูลที่เข้ากันได้ของโอเปอเรเตอร์ จะเกิดข้อผิดพลาดที่เรียกว่า Type error ซึ่งหมายถึงการใช้ชนิดข้อมูลที่ไม่เหมาะสม กระบวนการ Type checking อาจจะเกิดขึ้นในเวลาคอมไพล์หรือในเวลารันก็ได้ ตัวอย่าง Static variable จะเกิด Type checking ในเวลาคอมไพล์ และในภาษาที่มีการประกาศชนิดข้อมูลก่อนใช้งานจะทำ Type Checking ในเวลาคอมไพล์เช่นกัน 

static int x;
        int  z;
        x++;
        z++;

ถ้าภาษาการโปรแกรมไม่มีการประกาศชนิดข้อมูลให้กับตัวแปร การทำ Type checking จะเกิดในเวลารัน เช่นใน ภาษา PHP 

     $x = 10;
             $x++;            /* Can ++ be implemented by x */
             $x = “Hello”;     /* x is string */
             $x =   $x + 10;   /* Can  + be implemented by x */

ภาษาการโปรแกรมจะถูกเรียกว่าเป็น ถ้า type errors สามารถตรวจจับได้ก่อนเวลารัน เช่น ภาษา C และ Java เป็นต้น

int x, y; การประกาศตัวแปรทำให้ตัวแปร x และ y มีชนิดข้อมูลเท่าเทียมกันซึ่งเรียกว่า

ชนิดข้อมูลที่มีโครงสร้างเท่าเทียมกันเรียกว่า

นิพจน์ a + b เมื่อ a มีชนิดข้อมูลแบบ Integer และ b มีชนิดข้อมูลแบบ Floating point สามารถคำนวณร่วมกันได้เพราะชนิดข้อมูล....
