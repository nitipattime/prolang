ตัวแปรกับค่าข้อมูล

ในการเขียนโปรแกรมจะต้องทำการกำหนดตัวแปรขึ้นมาใช้งานเสมอ ตัวแปรเป็นตัวอ้างอิง (identifier) ที่ใช้ในการอ้างอิงข้อมูลที่อยู่ในหน่วยความจำหลัก ข้อมูลที่เก็บในหน่วยความจำหลักมี 2 ประเภท

พอยน์เตอร์ก็คือตัวแปร แต่เป็นตัวแปรที่เก็บค่าข้อมูลประเภทที่ 2

อธิบายโค้ด


ตัวแปร ถูกกำหนดด้วยคำที่หลากหลาย ซึ่งมีความแตกต่างในการนิยามไปตามสิ่งที่เกี่ยวข้องในลักษณะต่างๆ ซึ่งในภาษาส่วนใหญ่จะกล่าวถึงตัวแปร 5 ประเภทดังต่อไปนี้
  • หมายถึงตัวแปรที่สร้างขึ้นในพื้นที่หน่วยความจำที่เรียกว่าสแตก (stack)
  • หมายถึงตัวแปรที่อ่านค่าได้อย่างเดียวไม่สามารถแก้ไขค่าได้
  • หมายถึงตัวแปรที่มีขอบเขตภายในหน่วย (Unit) (อาจเป็นฟังก์ชันหรือคำสั่งควบคุม)
  • หมายถึงตัวแปรที่ผูกมัดกับตำแหน่งในหน่วยความจำเพียงตำแหน่งเดียวตั้งแต่กระบวนการคอมไพล์และคงอยู่ไปตลอดเวลารัน

ลองใช้โค้ดภาษา C เพื่อศึกษา ตัวอย่างการใช้งานตัวแปร Stack Variable และ Pointer Variable 

#include 
        int main()
        {
        
        int *ptr, q;
        q = 50;
        /* address of q is assigned to ptr */
        ptr = &q;
        /* display q's value using ptr variable */
        
        printf("Q has value %d. It is allocated at the address %p.\n", q, &q);
        
        printf("Pointer ptr points to value %d. It keeps the address %p of q. But it is allocated in %p", *ptr, ptr,&ptr);
        
        return 0;
        
        }

อธิบายโค้ด


Memory Layout ในภาษา C [https://www.geeksforgeeks.org/memory-layout-of-c-program/] ดัวอย่างการจัดการหน่วยความจำในภาษา C ประกอบด้วยพื้นที่ต่างๆ ดังต่อไปนี้ Text segment หรือ Code segment เป็นบริเวณจัดเก็บโค้ดคำสั่งของโปรแกรม Initialized data segment พื้นที่เก็บตัวแปรที่มีการประกาศชนิดตัวแปรและมีการกำหนดค่าเริ่มต้น เช่น int x = 1; , static int count = 0; Uninitialized data segment พื้นที่เก็บตัวแปรที่มีการประกาศชนิดตัวแปรแต่ไม่มีการกำหนดค่าเริ่มต้น เช่น int x;, static int count ; Heap segment พื้นที่สำหรับการทำ Dynamic memory allocation เช่น เมื่อใข้คำสั่ง malloc Stack เป็นพื้นที่สำหรับจัดเก็บค่าข้อมูลของตัวแปรต่างๆ รวมไปถึง pointer อนึ่งการจัดการหน่วยความจำในภาษาการโปรแกรมอื่นๆ หรือภาษาเดียวกันแต่เวอร์ชันต่างกัน ก็อาจจะแตกต่างกันไป ขึ้นกับการออกแบบของภาษาการโปรแกรม

        int main()
