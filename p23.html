ตัวแปรกับ

ตามที่ได้อธิบายในตอนที่ผ่านๆมา ตัวแปรมักจะถูกเรียกในลักษณะต่างๆตามปัจจัยที่นำมาจำแนก เช่น จำแนกตามหน่วยจัดเก็บข้อมูล จำแนกตามการจัดการขอบเขต เช่นตัวแปรสตาติก และนันสตาติก จำแนกตามหน่วยจัดเก็บข้อมูล การจำแนกตัวแปรตามการจัดการขอบเขต สามารถกำหนดได้ 2 ประเภท

คือ ตัวแปรที่มีการประกาศใช้งานภายในหน่วย (Unit) ซึ่งอาจเป็นภายในฟังก์ชันหรือบล็อค (Block) (เช่นภายในโครงสร้างควบคุม ลูป หรือ ประโยคเงื่อนไข) หรือหน่วยโปรแกรม ดังนั้นการมองเห็นตัวแปรโลคอลจะสัมพันธ์กับหน่วย (unit) เสมอ

คือ ตัวแปรที่มีการประกาศใช้งานอยู่ภายนอกหน่วย (โปรแกรม/ฟังก์ชัน/เมธอด) หรือ บล็อค แต่สามารถมองเห็นได้จากหน่วยหรือบล็อคนั้น

นอกจากนี้แล้ว ยังมีคำที่ใช้เรียกตัวแปรอีกประเภทหนึ่งซึ่งปรากฎเสมอในภาษาการโปรแกรม นั่นคือ ซึ่งภาษาการโปรแกรมส่วนใหญ่มักนิยามว่าเป็นตัวแปรที่ประกาศที่หน่วยของโปรแกรม แต่การมองเห็นในหน่วยอื่นๆ นั้นมีข้อกำหนดแตกต่างกันไปในแต่ละภาษาการโปรแกรม สิ่งสำคัญที่ควรพิจารณาในเรื่องคือ กรณีที่หน่วยซ้อนหน่วยจะกำหนดการมองเห็นตัวแปรอย่างไร เช่น ฟังก์ชันซ้อนโปรแกรม และ บล็อกซ้อนบล็อก ในแต่ละภาษาการโปรแกรมอาจมีข้อกำหนดที่แตกต่างกันไป ตัวอย่างภาษา Java

public static void main(String[] args) {   //local scope
            int x = 1;
            // A local scope where args and x are visible
        
            while (x <= 5) {                      //block scope
                String msg = "x is now " + x;
                // A local scope where args, x, and msg are visible
                System.out.println(msg);
            }                                    //End block scope
            // msg is no longer in scope, and has been flushed from memory
        
        }

ตัวอย่างภาษา Python ที่มีลักษณะแบบฟังก์ชันซ้อนฟังก์ชัน ภาษา Python มีการประกาศตัวแปรแบบ Implicit declaration คือ การประกาศเกิดขึ้นเมื่อมีการกำหนดค่า เมื่อพิจารณาที่ฟังก์ชัน outside จะมี คือ msgout เมื่อพิจารณาที่ฟังก์ชัน inside จะมี คือ msgin และมี Nonlocal variable คือ msgout 

>>> def outside():
                msgout = "Outside!"
                def inside():
                    msgin = "Inside!"
                    print(msgin)
                    print(msgout)
                inside()
                print(msgout)
        
        >>> outside()     #เรียกฟังก์ชั่น outside มาทำงาน
